Jak přidat CSS

Externí CSS

  1. Vytvoříme složku CSS a v něm soubor styl.css
  2. V HTML souboru napojíme CSS soubor:
        
      <head>
        <link rel="stylesheet" type="text/css" href="/CSS/styl.css">
      </head>

Interní CSS

píšem do hlavičky HTML souboru

        
      <head>
        <style>
          kód css
        </style>
      </head>

Inline CSS

Píšem jako Atribut

        
          <div style="kod css;">

          </div>

Selektory

Selektory (select = označit)
Selektory CSS se používají k "vyhledání" (nebo výběru) elementů HTML, které chcete stylovat.
Selector HTML Element CSS soubor Popis
#ID <div id="zelena"></div> #zelena { CSS kód } Element s jedinečným "ID".
.class <div class="cervena"></div> .cervena { CSS kód } Všechny Elementy s class="cervena" (Název class nemůže začínat číslem!).
element.class <div class="modra"></div> div.modra { CSS kód } Všechny Elementy div s class="modra".
element <div></div> div { CSS kód } Všechny Elementy div.
element, element, element <div></div><h1></h1><p></p> div,h1,p { CSS kód } Více Elementů
* * { CSS kód } Všechny elementy HTML

Barvy (Color)

Dle názvu barev:

w3schools/colors

RGB (červená (red), zelená (green) a modrá (blue))

Každý parametr (červený, zelený a modrý) definuje intenzitu barvy mezi 0 a 255.

background-color: rgb(255, 0, 0);

RGBA (červená (red), zelená (green), modrá (blue) a alfa)

Alfa parametr je číslo mezi 0,0 (plně transparentní) a 1,0 (vůbec netransparentní):

background-color: rgba(255, 0, 0, 0.3);

HEX - #rrggbb (rr (červená), gg (zelená) a bb(modrá)

Hexadecimální čísla se zapisují pomocí číslic '0', '1', '2', '3', '4', '5', '6', '7', '8' a '9' a písmen 'A', 'B', 'C', 'D', 'E' a 'F'

background-color: #ff0000;
