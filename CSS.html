CSS

CSS (Cascading Style Sheets)
Deklarativní jazyk, který určuje, jak vypadají webové stránky v prohlížeči.

CSS styluje elementy HTML, SVG nebo XML.

CSS poprvé implementovala společnost microsoft v roce 1996 do internet Exploreru 3.0.

https://www.w3.org/Style/CSS/

Jak přidat CSS

Externí CSS

  1. Vytvoříme složku CSS a v něm soubor styl.css
  2. V HTML souboru napojíme CSS soubor:
        
      <head>
        <link rel="stylesheet" type="text/css" href="/CSS/styl.css">
      </head>

Interní CSS

Píšem do hlavičky HTML souboru.

        
      <head>
        <style>
          kód css
        </style>
      </head>

Inline CSS

Píšem jako Atribut.

        
          <div style="kod css;">

          </div>

Selektory

Selektory (select = označit)
Selektory CSS se používají k označení elementů HTML, které chcete stylovat.
Identifikátor (#ID)
ID má jedinečný název a můžem ho přiřadit jenom jednomu HTML elementu.
Třídy (class)
Class má jedinečný název a můžem ho přiřadit k více HTML elementů.
Název HTML Element CSS soubor Popis
#ID <div id="zelena"></div> #zelena { CSS kód } Element s jedinečným "ID".
.class <div class="cervena"></div> .cervena { CSS kód } Všechny Elementy s class="cervena" (Název class nemůže začínat číslem!).
element.class <div class="modra"></div> div.modra { CSS kód } Všechny Elementy div s class="modra".
element <div></div> div { CSS kód } Všechny Elementy div.
element element <div><h1></h1></div> div h1 { CSS kód } Potomek elementu div (h1).
element + element <div><h1></h1><p></p></div> h1 + p{ CSS kód } Elemtn h1 + p.
element, element, element <div></div><h1></h1><p></p> div,h1,p { CSS kód } Více Elementů
* * { CSS kód } Všechny elementy HTML
:hover <a></a>, <div></div>,... a:hover { CSS kód}, div:hover { CSS kód },... Po najetí myší na daní element
:active <a></a>, <div></div>,... a:active { CSS kód}, div:active { CSS kód },... Po zmáčknutí myší na daní element
:visited <a></a> a:visited { CSS kód} Po zmáčknutí myší na element a. Povolené styly jsou: color, background-color, border-color, outline color, column-rule-color, the color parts of fill and stroke.

Jednotky

Desetinou hodnotu odělujeme tečkou (.).

V CSS se nepíšou mezery.

Pokud nastavujem více hodnot, platí pořadí top-right-bottom-left a jednotlivé hodnoty se oddělují mezerou (margin: 50px 150px 111px 333px).

Pokud zapíšete jen číslo bez jednotky, prohlížeč ho bude pokládat za pixely.

Absolutní délky

Jednotky absolutní délky jsou pevné.

Jednotky absolutní délky se nedoporučují používat na obrazovce, protože velikosti obrazovky se tolik liší. Lze je však použít, pokud je známé výstupní médium, například pro rozložení tisku.

Jednotka Názec Převod V px
mm milimetery 1mm = 3.78px
cm centimetry 1cm = 10mm 1cm = 37.8px
in palec (inch) 1in = 2,54cm 1in = 96px
px pixels (1px = 1/96th of 1in) - každé zařízený má jinačí počet px
pt typografický bod (points) (1pt = 1/72 of 1in)
pc pica (picas) nebo též cicero (1pc = 12 pt)

Relativní délky

Jednotky relativní délky určují délku vzhledem k jiné vlastnosti délky.

Jednotky relativní délky se lépe škálují mezi různými vykreslovacími médii.

Barvy (Color)

Dle názvu barev:

w3schools/colors

RGB (červená (red), zelená (green) a modrá (blue))

Každý parametr (červený, zelený a modrý) definuje intenzitu barvy mezi 0 a 255.

background-color: rgb(255, 0, 0);

RGBA (červená (red), zelená (green), modrá (blue) a alfa)

Alfa parametr je číslo mezi 0,0 (plně transparentní) a 1,0 (vůbec netransparentní):

background-color: rgba(255, 0, 0, 0.3);

HEX - #rrggbb (rr (červená), gg (zelená) a bb(modrá)

Hexadecimální čísla se zapisují pomocí číslic '0', '1', '2', '3', '4', '5', '6', '7', '8' a '9' a písmen 'A', 'B', 'C', 'D', 'E' a 'F'
