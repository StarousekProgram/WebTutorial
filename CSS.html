CSS

CSS (Cascading Style Sheets)
je jazyk, speciálně vyvinutý pro stylování HTML

Vytvoříme složku CSS a v něm soubor styl.css

V HTML souboru napojíme CSS soubor:
<head>
    <link rel="stylesheet" type="text/css" href="/CSS/styl.css">
</head>
