Elementy (Tagy)

<tag> - Prázdný element

<tag></tag> - Element HTML

<tag atribut = "hodnota"> - Atribut je doplňující informace k tagu.

HTML element

Element HTML je definován počátečním tagem, nějakým obsahem a koncovým tagem

Element HTML lze vnořit (to znamená, že prvky mohou obsahovat další prvky).

Zákaldní Elementy

Tag Popis Atribut
<!DOCTYPE html> Definujeme že tenhle dokumentu je HTML5
<html></html> Obaluje celý HTML dokument manifest, lang, xmlns
<head></head> Obsahuje informace pro prohlížeč
<meta> Metadata - data která nejsou určeni pro uživatele, ale pro prohlížeče charset="utf-8"
name="author" content="Jméno autora"
name="description" content="Popis stránky pro vyhledávače"
http-equiv="refresh" content="30"
<title></title> Povinný tag, specifiguhe titulek stránky, SOE
<body></body> Definuje tělo dokumentu a slouží jako kontejner pro všechny viditelné obsahu, jako jsou nadpisy, odstavce, obrázky, hypertextové odkazy, tabulky, seznamy, atd
<h1><h2> - <h6> Nadpis </h1></h2> - </h6> Headings = Nadpis, <h1>definuje nejdůležitější nadpis, <h6>definuje nejméně důležitý nadpis.
<p> text </p> paragraph = odstavec
<pre> text </pre> Text je zobrazen s pevnou šířkou, zachovává mezery a konce řádků.
<hr> Horizontal Rules = Horizontální čára
<br> Line Breaks = zalomení řádku

Elementy na formátování HTML

Tag Popis Atribut
<b> text </b> Tučné zvýraznění textu (není důležitý text)
<strong> text </strong> Strong emphasis = Tučné zvýraznění textu (důležitý text)
<i> text </i> Italic = Kurzíva (technický termín, fráze z jiného jazyka, myšlení, jméno lodi, atd)
<em> text </em> Emphasis = Zvýraznění textu (Program pro čtení z obrazovky vysloví slova s důrazem, přičemž použije slovní přízvuk.)
<small> text </small> Smaller = Menší text
<mark> text </mark> Mark = Označéní nebo zvýraznění textu
<del> text </del> Delete = Text který byl z dokumentu odstraněn.
<ins> text </ins> Insert = Text který byl vložen do dokumentu.
<sub> text </sub> Subscript = Dolní index (H 2 O)
<sup> text </sup> Superscript = Horní index (a 2 )
<u> text </u>
<s> text </s> strike = přeškrtnutý textu
<a> href="Cíl odkazu"odkaz</a> anchor = zakotvit, připoutat
<img src="Zdrojový soubor" alt="alternativní text"> image = obrázek width, height

Tabulka

Tag Popis Atribut
<Table> tabulka </table> Začátek a konec tabulky
<thead> hlavička tabulky </thead> Popis hodnot co znamenají ve sloupcích
<tbody> tělo tabulky </tbody> Obsah tabulky
<th> popis tabulky </th> table header = nadpis tabulky (píšeme místo <td>)
<tr> řádek tabulky</tr> table row = řádek tabulky
<td colspan/rowspan="2"> buňka tabulky </td> table data colspan/rowspan="počet buňek" - slučuje buňky v: řádku tabulky/sloupci tabulky

Vzor

<table>
    <thead>
        <tr>
            <th>Nadpis tabulky</th>
            <th>Nadpis tabulky 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>obsach buňky</td>
            <td>obsach buňky 2</td>
        </tr>
    </tbody>
</table>

Seznamy

Tag Popis Atribut
<ul></ul> Unordered List = Neuspořádaný seznam
<ol></ol> Ordered List = Uspořádaný seznam reversed, start, type
<li></li> List Item = Položka seznamu value

</ul>

Slovníček pojmů (Definition List)

Tag Popis Atribut
<dl></dl> Definition List = Slovníček pojmů
<dt></dt> Definition Term = Vysvětlovaný pojem
<dd></dd> Definition Definition = Vysvětlený pojmu

Vzor
