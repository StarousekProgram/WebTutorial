Struktura HTML stránky

<!DOCTYPE html> - definujeme, že textový soubor je HTML dokumentem
<html lang="cs-cz"> - zde začíná HTML stránka (lang definuje jazyk stránky)
    <head>
        <meta charset="utf-8" /> - kódování stránky
        <title>text titulku</title>
    </head>

    <body>
    </body>
</html>
