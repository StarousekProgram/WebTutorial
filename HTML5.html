Tagy

<tag> - nepárový tag

<tag></tag> - párový tag

<tag atributy> - Atribut je doplňující informace k tagu.

Elementy v HTML se dělí na řádkové a blokové. Rozdíl mezi nimi je ten, že blokové mohou v sobě obsahovat oba druhy, ale řádkové mohou obsahovat jen řádkové. Odkaz je element řádkový, stejně jako všechny elementy, co jsme si zatím zmínili, kromě nadpisů, které jsou blokové. Do odkazu tedy můžeme dát klidně i obrázek, ale ne nadpis.

Seznam tagů

Tag Popis Atribut
<!DOCTYPE html> Není ve skutečnosti tag
Nastavíme typ dokumentu na HTML jazyk
<html></html> Obaluje celý HTML dokument manifest, lang, xmlns
<head></head> Obsahuje informace pro prohlížeč
<meta> Metadata - data která nejsou určeni pro uživatele, ale pro prohlížeče charset="utf-8"
name="author" content="Jméno autora"
name="description" content="Popis stránky pro vyhledávače"
http-equiv="refresh" content="30"
<title></title> Povinný tag, specifiguhe titulek stránky, SOE
<body></body> Obsahuje fyzický obsah dokumentu
<h1><h2> - <h6> Nadpis </h1></h2> - </h6> až 6 nadpisů
<p> text </p> paragraph = odstavec
<em> text </em> emphasis = zvýraznění textu (důležitý text)
<strong> text </strong> strong emphasis = silné zvýraznění textu (důležitý text)
<u> text </u> podtržení textu
<s> text </s> strike = přeškrtnutý textu
<mark> text </mark> optickému zvýraznění textu
<img src="" alt=""> image = obrázek src, alt - název obrázku (Google Images)
<a> href=""odkaz</a> anchor = zakotvit, připoutat href

Tabulka

Tag Popis Atribut
<Table> tabulka </table> Začátek a konec tabulky
<thead> hlavička tabulky </thead> Popis hodnot co znamenají ve sloupcích
<tbody> tělo tabulky </tbody> Obsah tabulky
<th> popis tabulky </th> table header = nadpis tabulky (píšeme místo <td>)
<tr> řádek tabulky</tr> table row = řádek tabulky
<td colspan/rowspan="2"> buňka tabulky </td> table data colspan/rowspan="počet buňek" - slučuje buňky v: řádku tabulky/sloupci tabulky

</table>
